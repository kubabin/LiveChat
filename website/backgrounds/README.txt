Put your screenshots here with these exact filenames:

  default.jpg   - shown when nothing is hovered
  season1.jpg   - shown when hovering "Season 1"
  season2.jpg   - shown when hovering "Season 2"

.png also works (just keep the same base names: default.png, season1.png, season2.png)
but don't mix formats — the page checks for .jpg first, so if you use .png,
update the `background-image` paths in landing.css to match (see comments there).

Recommended: 1920x1080 or larger, since they're blurred/darkened and cropped
to fill the screen — small images will look soft at large screen sizes.
